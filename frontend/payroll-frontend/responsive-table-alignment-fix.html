Technical Implementation

The fix uses CSS table-layout: fixed and specific column width constraints to ensure consistent alignment.

CSS Classes Added:

  • .table-col-employee - Auto width with 200px minimum
  • .table-col-currency - Fixed 120px width for monetary values
  • .table-col-actions - Fixed 80px width for action buttons

Responsive Behavior:

  • Desktop (>768px): Full width columns for optimal spacing
  • Mobile (≤768px): Reduced column widths to fit smaller screens
  • All sizes: Content remains perfectly centered
